Detailed Description

PID-styret DC-motor med encoder-feedback på Arduino.

Roterer en DC-motor et bestemt antal grader ved hjælp af en PID-regulator og en quadrature-encoder til positionsfeedback.

void dc_motor_update ( )

PID-regulator – skal kaldes gentagne gange fra loop().

Beregner motorkommando ud fra positionen:

  1. Returnér tidligt hvis målet allerede er nået
  2. Beregn fejl = target − nuværende position
  3. Stop hvis fejlen er inden for threshold (dødzone)
  4. Opdater integral med anti-windup (clamp ±1000)
  5. Beregn differentialterm
  6. Beregn PID-output og send til motor via turn()

◆ encoderISR()

void encoderISR ( )

Interrupt Service Routine – opdaterer encoder-positionen.

Kaldes automatisk ved RISING edge på encoder1 (kanal A). Retningen bestemmes ved at aflæse kanal B:

  • Kanal B HIGH → med uret → tæl op
  • Kanal B LOW → mod uret → tæl ned

void stop_motor ( )

Bremser motoren og nulstiller PID-tilstand.
