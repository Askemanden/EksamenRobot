Measures distance using an ultrasonic sensor. 

Sends a short trigger pulse and measures the duration of the returning echo pulse. The duration is converted to distance in centimeters using the formula:\+

distance\+\_\+cm = duration /\+ 58

\begin{DoxyReturn}{Returns}
Distance to the nearest object in centimeters.
